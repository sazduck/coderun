На роботизированном производстве конвейер выпускает детали в порядке убывания их качества. Каждая деталь имеет числовой показатель качества — чем меньше число, тем хуже качество детали.

К сожалению, из-за технической неисправности в какой-то момент времени конвейер начал производить только дефектные детали. Известно, что как только появилась первая дефектная деталь, все последующие детали тоже стали дефектными.

Деталь считается дефектной, если её показатель качества строго меньше заданного порогового значения k.

Вам необходимо найти количество качественных деталей, которые успели произвести до появления первой дефектной детали.

## Формат ввода

Первая строка содержит два целых числа n и k (1 ≤ n ≤ 2·10⁵, 1 ≤ k ≤ 10⁹) — количество деталей на конвейере и пороговое значение качества соответственно.

Вторая строка содержит n целых чисел a₁, a₂, ..., aₙ (1 ≤ aᵢ ≤ 10⁹) — показатели качества деталей в том порядке, в котором они сошли с конвейера. Гарантируется, что массив отсортирован по невозрастанию: a₁ ≥ a₂ ≥ ... ≥ aₙ.
## Формат вывода

Выведите одно целое число — количество качественных деталей (с показателем качества ≥ k).

## Применчание

В первом примере пороговое значение k = 50. Детали с показателями 100, 80, 60 имеют качество ≥ 50, а детали с показателями 40, 30, 20 — дефектные (< 50). Ответ: 3.

Во втором примере пороговое значение k = 25. Качественные детали: 90, 70, 50, 30. Только последняя деталь (10) дефектная. Ответ: 4.

В третьем примере все детали дефектные, так как ни одна не достигает порогового значения 100. Ответ: 0.

В четвёртом примере все детали качественные. Ответ: 3.
